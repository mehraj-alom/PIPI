Why this system

Patients usually need a clean upload journey, not raw technical output. This interface focuses on clarity first: submit quickly, see confirmation immediately, and reveal deeper analysis only when needed.

What it does

Runs skin image detection with explainability, parses medical documents, and supports voice-agent appointment workflows.

How results appear

By default, patients only see a submission summary and the detected region image. Sensitive labels stay hidden behind See Result.

When to trust outputs

All outputs are educational support. Lower confidence or unclear findings should always be validated by a licensed clinician.

How to use

  1. Enter patient and session details to keep all files grouped under one case.
  2. Upload either a skin image or a document from the cards above.
  3. Review the default submission card, then tap See Result only when detailed output is needed.
  4. Use the call button to open the voice assistant for patient lookup and appointment actions.

PIPI is an educational clinical support interface and does not replace professional medical judgment.

Use this system to organize submissions, surface explainability, and accelerate care coordination.
